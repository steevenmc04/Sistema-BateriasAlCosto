# Sistema Baterías al Costo — UI Rules

Proyecto React + Vite + Tailwind.

Reglas obligatorias:
- No modificar backend, endpoints, hooks ni lógica de negocio sin permiso.
- Solo UI/UX cuando el usuario pida diseño.
- Usar paleta negro/dorado:
  - bg-black
  - bg-zinc-950
  - border-border-default
  - text-text-primary
  - text-text-muted
  - text-success
  - text-warning
  - text-error
- No usar selects nativos.
- Usar SelectPremium para desplegables.
- Usar input-premium para inputs.
- Usar card-premium para cards.
- Usar table-premium para tablas.
- Usar modal-premium para modales.
- Todo responsive.
- Ejecutar npm run build después de cambios grandes.
